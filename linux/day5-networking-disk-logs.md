## Networking Basics
- IP address identifies a machine on a network
- ping checks connectivity
- 127.0.0.1 is loopback address

## Ports
- Ports allow multiple services on one server
- Common ports: 22 (SSH), 80 (HTTP), 443 (HTTPS)

## Disk Management
- df -h shows disk usage
- du -sh checks directory size
- Disk full can crash applications

## Logs
- Logs record system and application events
- First place to check during failures
- tail -f is used for real-time log monitoring

## Monitoring Mindset
- Monitor CPU, memory, disk and network
- Proactive monitoring prevents outages
