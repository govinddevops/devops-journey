# Day 2 – Linux Basic Commands

## 1. pwd (Print Working Directory)
Current directory ka full path dikhata hai.
```
pwd
```

---

## 2. ls (List)
Directory ke andar kya files/folders hain ye dikhata hai.
```
ls
ls -l
ls -a
```
- `-l` → details ke saath list
- `-a` → hidden files bhi show karta hai

---

## 3. cd (Change Directory)
Directory change karne ke liye use hota hai.
```
cd linux
cd ..
cd ~
```

---

## 4. mkdir (Make Directory)
Naya folder banane ke liye.
```
mkdir test
mkdir devops linux
```

---

## 5. touch
Empty file create karne ke liye.
```
touch file.txt
touch day2-commands.md
```

---

## 6. cat
File ka content dekhne ke liye.
```
cat file.txt
```

---

## 7. nano
Terminal ke andar file edit karne ke liye.
```
nano file.txt
```
Save: Ctrl + O  
Exit: Ctrl + X

---

## 8. rm (Remove)
File ya folder delete karne ke liye.
```
rm file.txt
rm -r foldername
```
⚠️ Warning: Delete permanently hota hai.
