# Monitoring & Observability Stack

This project demonstrates a professional-grade monitoring setup. It is a key part of my DevOps journey to ensure infrastructure stability and performance.

## 🏗️ Architecture
- **Prometheus:** Scrapes time-series data from servers.
- **Grafana:** Creates visual dashboards for observability.

## 🚀 Skills Implemented
- Local monitoring agent configuration.
- Service management with systemctl.
- Dashboard importing and Data Source linking.
